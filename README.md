# 面经汇总

---

## 前言

本文主要收录了一些关于前端方面的面试题以及个人整理的一些笔记。其中，大部分文章来自于[掘金](https://juejin.cn/)，少部分来自[思否](https://segmentfault.com/) 、[CSDN](https://www.csdn.net/) 等博客平台。

由于前端体系涉及内容繁杂，面试题涵盖的方面较多，推荐先阅读这篇文章：[2021年前端面试必读文章【超三百篇文章/赠复习导图】](https://juejin.cn/post/6844904116339261447) 。若对前端知识体系有了初步的了解，可针对每个方面的知识进行学习，掌握对应的面试题目。

本文共分为了三个模块：精华版、详尽版和个人版。其中，精华版是收录的一些面试题汇总，其中一些问题的答案较为简略，需要读者另外收集一些资料进行学习；详尽版是针对前端体系的各个方面的一些面经汇总，内容较为详细；个人版是作者的一些笔记，较为杂乱，若不敢兴趣可忽略。
